## 👉 서비스 소개
* 밤정리는 어떤앱인가요?<br>
  * 하루를 되돌아 보는 시간과 함께 주변 정리를 도와주는 방정리 서비스 앱 입니다.<br>
* 왜 밤에 정리해야하나요?<br>
  * 밤에는 아세틸콜린이 활성화되기 때문이죠. 아세틸콜린은 세포의 상호소통을 도와주며 기억, 학습, 집중에 도움을 준다고 합니다.<br>
  * 무기력할 때 의욕이 생길 때까지 기다리기보다는 '일단 시작하자'라고 생각하고 바로 하는 것이 굉장히 중요하기 때문입니다.<br>
